# C++还原"超级马里奥"小游戏  
* 本项目是基于 C++ 与 EasyX 图形库对超级马里奥游戏进行的还原仿制  
* 包括了马里奥游戏的基本玩法内容  
* 目前已制作出完整的 1-1,1-2,1-3 三个关卡  
* 通关 1-3 后程序自动关闭属正常现象  
# 按键操作  
* 移动 a/d  
* 跳跃 k  
* 加速/发射火球 j  
* 下蹲/钻入管道 s  
# 编译环境  
* [Microsoft Visual Studio 2022](https://visualstudio.microsoft.com/)  
* [EasyX_20220901](https://easyx.cn/)
